# 说明
1. 本帖整理自《临高汉词典》
2. 本帖只收录临高话的汉字读书音，不收老借词音，也不收海南话、儋州话、粤语、普通话借音
3. 阳平和上声少数字互混，可能是受到海口话影响（海口话阳平31接近临高话汉字音上声21，海口话上声213接近临高话汉字音阳平13），此类字标*
4. 个别字声韵属老借词音，声调属读书音，标<>
5. 其他读音特殊的字标[]
# 声调
> 1阴平(33) 2阳平(13) 3上声(21) 4去声(55) 5入声(55)
# 特点
1. 见系戈韵低化回ua，但端系歌戈不能分，说明端系戈韵死得异常早
2. 影母完全独立，喻母读j-、v-
3. 疑母开口一三四等都不挂，但二等有一部分读j-。然而这一部分字也不混喻母：雅ja!=惹jia，眼jan!=衍jian。疑母合口呼也不挂
4. 覃韵、合韵有残留
5. 山摄见二开全无介音，但疑母读j-暴露了其介音失落是后起的。可以参考粤语的ɐ类韵。
